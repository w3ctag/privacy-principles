Jeffrey's highlight

2.8 Device Owners and Administrators

  • Devices are owned by people other than their users
  • UAs have to balance these interests
  • Owners can ensure reasonable restrictions on use
  • Users must know about data disclosure
Robin's highlight

1.3 User Agents

  • "The user agent is expected to align fully with the person using it and to operate exclusively in that person's interest."
  • Duties of protection, discretion, honesty, and loyalty
  • Trustworthy user agents provide a fiduciary framework for privacy, which has a number of advantages over other approaches
